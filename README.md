# Restaurant Ordering App

A simple, interactive restaurant ordering application built with HTML, CSS, and JavaScript.  
The app displays available menu items, lets users add or remove items from their order, calculates the total bill, and simulates a checkout/payment process.

---

## Features

- Displays a dynamic menu using data from a JavaScript module.
- Add items to an order with a single click.
- Remove individual items from the order.
- Automatically calculates total order cost.
- Includes a mock payment section with form validation.
- Shows a success message after payment is completed.
